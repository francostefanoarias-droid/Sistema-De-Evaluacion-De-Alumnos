// Sistema de evaluación de alumnos

Sistema desarrollado en C# el cual permite ingresar datos de un alumno, calcular su promedio y determinar su estado académico.

// Funcionalidades

-- Validación de nombre (no se permite la utilización de números, simbolos, etc)
-- Ingreso de 3 notas (0 a 10)
-- Cálculo automático del promedio
-- Clasificación del alumno:
  -- Desaprobado
  -- Aprobado
  -- Promocionado

// Ejecución

1. Ejecutar el programa
2. Ingresar nombre y notas
3. Visualizar resultados
